IMPACT Attestation Tool

Welcome to the IMPACT Attestation Tool.
This tool enables you to attest words by higlighting them in citations.

Click here to log in.

In the screenshot below you see an example page containing five quotations for the Dutch word 'ruiten'.

  • Quotations are displayed per lemma. They are ordered by degree of uncertainty about their correctness.
    The most uncertain citation is displayed at the top and its color is the most red. Literal citatations are green.
  • If there is something wrong with the citation that you can not fix with this tool, mark it as a 'bad' citation by clicking the so it becomes .
  • If the current headword doesn't really occur in de citation, mark this by changing the into a .
  • By hitting the button, the word that was attested last will be automatically attested in the rest of the page.
  • You can delete the selected attestation by clicking on it or by hitting the DELETE button on your keyboard.
  • You can add an attestation by clicking on a word, or by hitting the INSERT button on your keyboard (which will mark the first non-attested word as a new attestation).
  • You can move the selected attestation around by using the arrow keys (←, ↑, →, ↓) on your keyboard.
  • You can make a group of attestations just by holding the mouse down while selecting the words in the group. By holding down the appropriate key for LOCATION/ORGANISATION/PERSON>NOT KNOWN the group will be attested as such. If you don't hold down a key while selecting the group it will be a PERSON by default.
Keys:
F2 or d Previous revised lemma.
F4 or f Next revised lemma.
Spacebar Save current lemma, and display a new unrevised one.
F8 Type in some comment.
F9 or x Toggle quote as good/bad .
u Toggle as fortunate/unfortunate citation .
a Auto attest .
e (Un)mark attestation as elliptical (e.g. North and South America).
s (Un)mark attestation as erroneous.
w (Un)mark attestation as dubious.
INSERT Insert a new attestation.
DELETE Delete currently selected attestation.
< > Walk through attestations of the selected quote.
\ Split the token.
` (backquote) Hold this key down while clicking on an attestation to start a group attestation. `-clicking on other tokens will add them to the group.
1 Normally if an attested token is clicked on it will become unattested. If you hold the '1' key while clicking it will just be selected.

In the case of multiple typed attestations for named entities there are several more keys:
m   Pull down the type menu for the currently selected attestation.
l   If this key is held down while clicking on an attestation, it will be attestated as a LOCATION. Also if you press it when the type menu is open, the currently selected attestation will become a LOCATION.
n   If this key is held down while clicking on an attestation, it will be attestated as NOT KNOWN. Also if you press it when the type menu is open, the currently selected attestation will become NOT KNOWN.
o   If this key is held down while clicking on an attestation, it will be attestated as an ORGANISATION. Also if you press it when the type menu is open, the currently selected attestation will become an ORGANISATION.
p   If this key is held down while clicking on an attestation, it will be attestated as a PERSON. Also if you press it when the type menu is open, the currently selected attestation will become a PERSON.
